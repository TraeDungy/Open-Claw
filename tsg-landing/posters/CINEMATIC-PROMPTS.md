## USAGE NOTES

### For ChatGPT Image Gen 2 (gpt-image-1):
- Paste the prompt directly into ChatGPT
- Request "movie poster quality, photorealistic, cinematic"
- For vertical: ask for 1024x1536 or specify "portrait poster format"
- For horizontal: ask for 1536x1024 or specify "widescreen 16:9"

### For Midjourney:
- Add `--ar 2:3` for vertical, `--ar 16:9` for horizontal
- Add `--style raw` for photorealistic output
- Add `--v 6.1` for latest model
- Add `--q 2` for maximum quality

### For DALL-E 3:
- Paste prompt as-is
- Specify size: 1024x1792 (vertical) or 1792x1024 (horizontal)
- Add "photorealistic, cinematic, movie poster quality" if needed

### Typography Overlay:
After generating the image, overlay text in Canva/Photoshop/Figma:
- **Title font:** Rajdhani Bold or Big Shoulders Bold
- **Subtitle:** Jura Light, widely tracked
- **Brand:** GeistMono or DM Mono, 10pt
- **Colors:** Bone #F5E3B3 for titles, Ember #FF902E for accents, Signal #FFC260 for highlights
